# article-guide

디지털데일리 IT 기사 작성·검증 에이전트(`it-article-writer` 스킬)의 데이터 저장소.

## 파일 구성

| 파일 | 용도 |
|------|------|
| `guide.md` | 기사 작성 요령 (문체·구조·출처 우선순위·팩트체크 기준). 에이전트의 **단일 기준**. |
| `examples.md` | 기자별 대표 기사 본문 예시. **문체 참고 전용** (팩트 출처로 사용 안 함). |
| `article_list.json` | 백지영·채수웅·이안나·김보민 기자 기사 메타데이터 5,122편 (2025~2026). 문체 참고용. |
| `skill/SKILL.md` | 에이전트 정의 백업본. |

## 에이전트 동작 (2단계)

1. **작성(Phase 1)**: 취재 자료(텍스트/파일/URL) → `guide.md` 문체로 기사 초안. 사용자 확인 대기.
2. **검증(Phase 2)**: 자료 내부 일관성 + 웹 교차검증(위키 계열 제외) → 검증 리포트 + 최종 기사.

## 스킬 설치 위치

- 영구 보존: `~/.claude/skills/it-article-writer/SKILL.md`
- 백업: 이 레포 `skill/SKILL.md`
